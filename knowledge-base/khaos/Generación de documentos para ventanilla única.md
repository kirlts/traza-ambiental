# Generación de documentos para ventanilla única


## Qué es

Generación de planillas o paquetes documentales para declaraciones ante la ventanilla única del MMA. Declaración anual y potencialmente mensual. Sin acceso API al MMA; el usuario sube manualmente los documentos generados por el sistema.
