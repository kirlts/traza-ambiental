# Metas para Neumáticos Categoría B

## Qué dice

Establece las metas cuantitativas de recolección y valorización para neumáticos fuera de carretera OTR (Cat B), escalonadas progresivamente por período. Las metas para Cat B son típicamente menos exigentes en porcentaje pero aplican a volúmenes masivos (minería), lo que genera un impacto económico significativo.

## Por qué existe

Este concepto se deriva de [[Decreto Supremo 8]] porque las metas cuantitativas para Cat B son establecidas directamente por dicho decreto. Para Trazambiental, Cat B es el segmento de mayor valor comercial (mineras, puertos), y las metas del DS 8 son los indicadores que el software debe rastrear para estos usuarios premium.

## Lógica de descomposición

Este concepto es atómico y no requiere mayor descomposición.

## Relaciones Horizontales

Este concepto no posee relaciones horizontales directas modeladas en el sistema.

Decreto Supremo Nº 8, anexos con metas escalonadas para Categoría B.
