# Metas para Neumáticos Categoría A

## Qué dice

Establece las metas cuantitativas de recolección y valorización para neumáticos de uso general (Cat A), escalonadas progresivamente por período. Las metas cubren porcentajes mínimos de toneladas recolectadas y valorizadas respecto al total introducido al mercado.

## Por qué existe

Este concepto se deriva de [[Decreto Supremo 8]] porque las metas cuantitativas para Cat A son establecidas directamente por dicho decreto. Para Trazambiental, estas metas son los KPIs normativos que el software debe medir y visualizar para sus usuarios Cat A, y contra los cuales la SMA fiscaliza el cumplimiento.

## Lógica de descomposición

Este concepto es atómico y no requiere mayor descomposición.

## Relaciones Horizontales

Este concepto no posee relaciones horizontales directas modeladas en el sistema.

## Fuente original

Decreto Supremo Nº 8, anexos con metas escalonadas para Categoría A.
