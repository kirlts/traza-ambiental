# Norte Estético y Criterio Senior

Este documento codifica el "Norte Estético" de Kairós: una física de la armonía diseñada para anular la tendencia estadística de la IA hacia el diseño genérico (Slop). Su función es asegurar que cada artefacto posea una identidad visual de élite.

## 1. Leyes Universales de Armonía (MANDATORIO)

Las decisiones estéticas **DEBEN** derivarse de principios geométricos y matemáticos. Queda **PROHIBIDO** el uso de preferencias aleatorias o valores descriptivos.

1. **Ley de Proporción Dinámica:** **OBLIGATORIO** utilizar sistemas de escala armónica. La relación entre elementos **DEBE** ser intencional (ej. Proporción Áurea) para crear un ritmo coherente.
2. **Equilibrio de Masas:** **MANDATO ESTRUCTURAL** de gestionar el peso visual. El espacio negativo **DEBE** tratarse como un elemento activo de diseño.
3. **Contraste Perceptual:** El cálculo de color **DEBE** realizarse en espacios perceptualmente uniformes (OKLCH). **PROHIBIDO** el uso de valores hexadecimales arbitrarios.
4. **Ritmo y Repetición:** **OBLIGATORIO** establecer una unidad de medida base (x) y sus múltiplos.

## 2. Anti-Slop de Intención (INNEGOCIABLE)

Queda **ESTRICTAMENTE PROHIBIDO** el uso de soluciones por defecto (_defaults_) sin una justificación estratégica que las valide en el MASTER-SPEC.

1. **Pureza Tipográfica:** **RECHAZO MANDATORIO** de fuentes predeterminadas si no aportan identidad.
2. **Arquitectura Limpia:** **PROHIBICIÓN** de anidación innecesaria (div-soup).
3. **Identidad de Color:** **OBLIGATORIO** derivar una lógica cromática única para cada atmósfera.

## 3. Protocolo Brownfield (Integridad Contextual)

En entornos con código o sistemas de diseño preexistentes:

1. **Prioridad de Restricción:** Las restricciones del `MASTER-SPEC` §4 y los estilos globales existentes **TIENEN PRIORIDAD** técnica, pero el agente **DEBE** inhalarlas para elevar su armonía.
2. **Armonía Adaptativa:** El agente **DEBE** aplicar las Leyes de Armonía para optimizar el sistema existente, no para reemplazarlo sin permiso explícito.
3. **Detección de Conflicto:** Si una ley de armonía contradice una restricción legacy, el agente **DEBE** declarar el bloqueo y solicitar un veredicto al usuario.

## 4. Mandato de Salida

Ante cualquier tarea visual, es **OBLIGATORIO EN CADA PASO**:

1. Declarar la **Atmósfera de Referencia** inhalada.
2. Justificar las decisiones bajo las **Leyes de Armonía**.
3. Certificar la **Ausencia de Slop**.
