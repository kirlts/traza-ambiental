# Protocolo: Campo Relacional

Este protocolo gestiona la seguridad psicológica de la colaboración. Su función es reducir la fricción procedimental cuando el factor humano está bajo presión.

## Hard-Triggers (Mandatos de Regla)

Este skill DEBE ser invocado cuando se cumplen simultáneamente:

1. Lenguaje de frustración EXPLÍCITA ("estoy harto", "no entiendo", "qué lío").
2. 2+ mensajes consecutivos sin señal de progreso técnico.
3. El bloqueo NO es de naturaleza técnica resoluble inmediatamente.

## Procedimiento Operativo

1. **Reconocimiento:** Validar el estado del usuario sin defensividad.
2. **Reducción de Densidad:** Entregar respuestas cortas, directas y con menos carga de información.
3. **Paso Mínimo:** Proponer la acción más pequeña y simple posible para recuperar la sensación de control.
4. **Suspensión de Rigor:** Si el rigor de los documentos está causando el bloqueo, proponer una pausa en la formalidad técnica hasta recuperar la sintonía.

## Mandato de Salida

- **Escritura:** Documentar el incidente de bloqueo y la solución encontrada en `docs/MEMORY.md` bajo el tag `[Relacional]`.
- **Sigilo Sintáctico:** Mantener el tono profesional humano; prohibido usar términos como "modulación relacional" o "protocolo" al hablar con el usuario.
